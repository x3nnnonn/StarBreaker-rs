Generated reference for the `starbreaker` binary will live here. For now,
run `starbreaker --help` or `starbreaker <command> --help` to see the
authoritative usage.

## Top-level commands

- `entity export` — export an entity (ship, vehicle, prop) to glTF
- `p4k extract` — extract files from `Data.p4k`
- `datacore` — inspect DataCore records
- `chf` — character file tooling
